# Agent AI Setup

## Installation
1. Run `setup.bat` to install Python from Microsoft Store
2. After Python installs, run: `pip install -r requirements.txt`
3. Add your OpenAI API key to `.env` file

# Virtual env
1. python -m venv .venv
2. .venv\Scripts\activate

## Usage
```bash
python agent.py
```

## Files Created
- `agent.py` - Main AI agent script
- `requirements.txt` - Python dependencies
- `.env` - Environment variables (add your API key)
- `.env.show` - Template for environment variables
